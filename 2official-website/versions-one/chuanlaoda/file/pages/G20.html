G20联盟

随着国家经济形式的变化，水运物流行业受到严重冲击，传统的长江水运物流行业受到严重冲击：运 价大幅下跌各项成本大幅增加，风险管控能力低，融资贷款难，经营方式落后，企业竞争力弱，原有模 式已经无法适应新形势下的发展。在此航运寒冬下，航运G20联盟应运而生，航运G20联盟由船老大网 及20家大型航运单位联合发起成立。目前已有超过50家联盟成员，年营业额超过20亿元，运输货种覆 盖:粮谷、钢材，矿产、煤炭、纸品、建材、化工品等绝大多数种类，航线分布于整个长江干线及内河全 流域
航运G20联盟作为航运企业的大本营，为成员做好各项切实可靠的服务，通过整合联盟内部的各种资 源，实现资源的共享共赢。并为成员提供船老大网运输管理系统，可对船运的整个流程实现全方位、一 体化的操作和掌控，大幅提高工作效率，降低物流成本。航运G20联盟努力为成员保驾护航让航运企业 在如此艰险泥泞的前路上不再单打独斗，在如此低迷下行的市场中不再孤军奋战！来吧顺应航运互联网 的时代潮流，加入这个大家庭，借用联盟和船老大网平台的力量做大做强！
联盟宗旨：以降本增效、共享共赢为宗旨，提高联盟成员管控和抗风险能力、增强企业竞争力。
