公司简介

船老大网是国内领先的内河水运交易服务平台，集约海量货源和运力资源，以信息技术和互联网 思维做支撑，提高效率、降低成本，为物流参与各方提供专业、贴心的服务。船老大网总部位于 上海；2015年1月上线以来,相继获得中路资本天使轮投资、贝越资本A轮投资，并作为三家高端 航运服务机构之一落户陆家嘴；2016年9月，船老大网又成为陆家嘴航运互联网专业委员会发起 单位。截至2016年12月，船老大网注册船只达10000多条,数十家货代联盟成员遍布全国30多个 城市。在未来，船老大网将继续努力为航运参与各方提供更加优质、便捷的服务，在推动整个行 业信息化、标准化和智能化的道路上砥砺前行。
